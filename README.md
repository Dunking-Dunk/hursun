## 🎯 What Makes Hursun Special

Hursun stands out as a unique portfolio website that combines cutting-edge web technologies with artistic 3D visualizations. Built from the ground up with a custom vanilla JavaScript framework, it demonstrates both technical prowess and creative vision.

### 🌟 **Why Choose Hursun?**
- **Performance First**: Custom-built framework ensures optimal loading times and smooth interactions
- **Visual Impact**: Eye-catching 3D elements that leave lasting impressions
- **Professional**: Clean, modern design that appeals to recruiters and collaborators
- **Unique**: Stand out from template-based portfolios with custom animations and effects

### ⚡ Installation

```bash
# 1️⃣ Clone the repository
git clone https://github.com/Dunking-Dunk/hursun.git

# 2️⃣ Navigate to project directory
cd hursun

# 3️⃣ Install dependencies
npm install
# or with yarn
yarn install
```

### 🔧 Environment Setup

Create a `.env` file in the root directory:

```env
# Server Configuration
PORT=3000
NODE_ENV=development

# Contact Form (optional)
CONTACT_API_KEY=your_api_key_here
EMAIL_SERVICE=your_email_service

# Analytics (optional)
GOOGLE_ANALYTICS_ID=your_ga_id
```

### 🎯 Running the Application

```bash
# 🚀 Start development server
npm start

# 🏗️ Build for production
npm run build

# 🧪 Run tests
npm test
```

**🌐 Open your browser and navigate to:** [http://localhost:3000](http://localhost:3000)
